# Hello World!

Welcome to my new personal website. This is a minimalist blog built with **Next.js 14+** (App Router), **MDX**, and **Tailwind CSS**.

## Why this setup?

- **Performance**: Static site generation (SSG) for fast loading times.
- **Developer Experience**: Writing content in Markdown.
- **Styling**: Tailwind Typography for beautiful reading experiences.
- **Hosting**: Free hosting with GitHub Pages.
